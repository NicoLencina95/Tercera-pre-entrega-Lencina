
1) Todos los templates tienen herencia del template padre.html
2) Si nos dirigimos a la pagina de inicio http://127.0.0.1:8000/App1 vamos a poder visualizar la pagina de inicio
3) Cada sección Cliente, Producto, Vendedor o Entregable presenta un espacio propio para profundizar desarrollo al
cual se puede acceder simplemente clikeando en la sección deseada. 
4) Podemos añadir información mediante formulario a cada sección a traves del html de la sección seguida de la palabra 
Formulario. Por ejemplo http://127.0.0.1:8000/App1/clienteFormulario salvo en la opción de Productos donde el formulario
se carga desde la misma pagina http://127.0.0.1:8000/App1/productos gracias a que se habilitó para que todo se haga desde
un mismo html
5)Tambien hay una opción de busqueda adaptada a Producto a través del html http://127.0.0.1:8000/App1/busquedaProducto
